[[ ENG ]]

--- Description ---

This is my second low poly set, under the 'Office' theme, containing lots of stuff you would see on any random office out there, with some variants of the same objects in certain cases. You will find "incomplete" models, or missing parts. These are not missing, in order to optimize it, I deleted clones of the same parts (such as drawers), you will have to manually clone these and adjust them yourself in order to setup them correctly, you can mix and match different parts in order to create distinct variations of the same too, the set is modular after all!

It's a pretty simple set, tried to optimize it as much as possible, so one can use it for mobile, PC or whatever (it's low poly after all), and comes with the palette I used originally (along with another 5 different ones based on the original first one, to provide variation), which is a single 8x8 pixels image, you can just change the color pixels to whatever you want in order to create new ones. Tried to also set up models in a ready-to-use way, you should have no problems regarding the pivot position of each object, or any weird artifact.

[Notes]
* If you want to use it on Unity, for example, you may find it useful to tweak certain options when importing the image/texture in order to properly import it correctly.
* Certain models, easy to tell who, feature a second material, (such as the whiteboard or the PC). This was done this way so one can easily change what's displayed within the model (like a PC displaying some windows img, or a whiteboard featuring some graph or whatever) rather than just a solid color.





[[ ESP ]]

--- Descripción ---

Este es mi segundo set, esta vez bajo la temática de "oficina", contiene tipicas cosas que uno encontraría en cualquier oficina, incluyendo variantes de ciertos objetos. Encontrarás modelos "incompletos", como escritorios sin cajones, que realmente no estan incompletos, simplemente decidí eliminar clones del mismo objeto, deberás clonarlos y ajustarlos por tu cuenta (no es tan difícil, y puedes combinarlos para crear variantes distintas, dado que es un set modular).

Es un set bastante simple, low poly, que trate de modelar de forma optimizada. Utiliza una paleta de 8x8 pixeles (junto a otras 5 variantes), que puedes modificar para crear nuevas paletas. Algunos modelos fueron diseñados de forma que puedan manipularse de forma sencilla y animarse por separado, no deberías de encontrar ningún problema en termino de ajuste de pivot ni nada por el estilo.
